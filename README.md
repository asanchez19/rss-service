## Proceso de Build

```
# instalar dependencias
yarn

# Iniciar el servidor (http://127.0.0.1:3000)
npm start

# Se puede utilizar nodemon para recargar los cambios automaticamente
npm install -g nodemon
nodemon --exec npm start

# correr pruebas
npm test
```

## Correr

`npm start`
